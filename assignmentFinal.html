FINAL ASSIGNMENT: web portfolio + final project

This last assignment is in two parts.

First off, assemble all of your sketches from the semester into an online portfolio website hosted at github pages.

Please revisit the web dev portion of the syllabus if you don't remember how to do this!

You should have a navigation bar, a pulldown menu to navigate your sketches and an about page of some sort.

Have fun with the design!

The second half of your final project is a deep dive into a specific equation. Find a math formula that interests you visually and port it to p5.

Check back here for examples and ideas in a few weeks.
